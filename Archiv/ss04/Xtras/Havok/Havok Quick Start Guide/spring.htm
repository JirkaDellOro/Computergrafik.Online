Federn und Stoßdämpfer

Nun bewegt sich Ihre Szene nach den Gesetzen der Physik und Sie können sie steuern und mit ihr interagieren. Das Havok Xtra stellt dazu für einzelne Starrkörper spezielle Funktionen bereit. Die Position und Drehung von Starrkörpern kann entweder unmittelbar (mit Hilfe verschiedener Funktionen vom Typ "…MoveTo" oder indirekt (in der Simulation durch das Anwenden von Kräften, Impulsen oder Drehmomenten) beeinflusst werden.

Über diese Standardfunktionen hinaus bietet das Havok Xtra auch integrierte und einfach zu handhabende Bedingungen in Form von Stoßdämpfern und Federn. Federn haben eine Entspannungslänge und eine Elastizität und funktionieren wie Federn in der realen Welt. Stoßdämpfer kann man sich als eine Art Begrenzer vorstellen. Lineare Stoßdämpfer legen die Position eines Starrkörpers fest. Die Position kann auf einen bestimmten Punkt im Raum der Welt oder auf einen bestimmten Punkt relativ zu einem anderen Starrkörper festgelegt werden. Mit Hilfe der zweiten Möglichkeit können Sie beispielsweise eine Kette von Starrkörpern erstellen. Drehungsstoßdämper sollen die Drehung eines Starrkörpers begrenzen. Das heißt, dass beispielsweise die Ausrichtung eines Würfels derart begrenzt werden kann, dass er immer auf einer Ecke balancieren soll.

Nun können Sie die aktuelle Szene verändern, um sie interessanter zu machen und nicht einfach nur einen Würfel auf eine Kugel fallen zu lassen. Sie können einen linearen Stoßdämpfer hinzufügen, um die Position des Würfels festzulegen, eine Drehung des Würfels jedoch zuzulassen. Außerdem können Sie eine harte Feder hinzufügen, um die Kugel mit dem örtlich fixierten Würfel zu verbinden. Diese Feder kann so konfiguriert werden, dass die Kugel andauernd mit dem Würfel zusammenstößt und eine interessante, ungeordnete Wechselbeziehung entsteht.

on createDashpot me

worldPoint = w.model("TheBox").transform.position
dashpot = hk.makeLinearDashpot( "DashpotName", "TheBox", worldPoint )
dashpot.strength = 1000
dashpot.damping = 100

end

on createSpring me

spring = hk.makeSpring( "SpringName", "TheBox", "TheBall" )
spring.restLength = 3.5
spring.elasticity = 10000
spring.damping = 1

end

Mit Hilfe dieses Lingo-Skripts erstellen Sie den linearen Stoßdämpfer und die Feder. Fügen Sie diesen Skriptabschnitt zum Verhalten hinzu. Die beiden neuen Funktionen müssen nach dem Erstellen der physikalischen Objekte innerhalb von beginSprite aufgerufen werden.

Wenn Sie den Film jetzt abspielen, sollte er ungefähr wie oben dargestellt aussehen.
