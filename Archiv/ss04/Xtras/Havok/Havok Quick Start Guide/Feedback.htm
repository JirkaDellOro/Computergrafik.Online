Rückmeldungen

Das Havok Xtra liefert über die Position und Ausrichtung von Starrkörpern hinaus viele weitere Informationen. Sie können zu jedem Zeitpunkt die Geschwindigkeit eines Körpers und die Größe der Kraft, die auf ihn wirkt, abrufen. Sie können auch Informationen über das Zusammenprallen von Starrkörpern abrufen. Zuletzt soll zu Ihrer Beispielszene noch ein Abschnitt eines Lingo-Skripts hinzugefügt werden, der bei jedem Aufprall der Kugel auf den Würfel einen Ton wiedergibt. Der dazu erforderliche Code ist in zwei Abschnitte gegliedert. Zunächst müssen Sie dem Havok Xtra mitteilen, dass die Kugel und der Würfel aufeinander prallen sollen.

on beginSprite me

w = member( 3 )
hk = member( 4 )

w.resetWorld()
hk.initialize( w, 0.1, 1 )

createVisibleObjects()
createPhysicalObjects()
createDashpot()
createSpring()
hk.registerInterest( "TheBox", "TheBall", 10, 0, #collision, me )

end

Dieses Skript ist die Endversion der Funktion "beginSprite". In der fett dargestellten Zeile wird angegeben, dass ein Zusammenprall erfolgen soll. Wenn ein Zusammenprall der Objekte "TheBox" und "TheBall" erfolgt, wird der zweite Code-Abschnitt (die Funktion collision) aufgerufen. Das Skript für die Funktion "collision" ist im Folgenden aufgeführt.

Hinweis: Bei diesem Code wird vorausgesetzt, dass die wiederzugebende Tondatei als Filmdarsteller "Sound" gespeichert ist.

on collision me, details

sound(1).volume = integer( 255 / power( details[5], 1.1 ) )
sound(1).play( member( "Sound" ) )

end

Wie aus dem Skript ersichtlich ist, kann bei dieser Callback-Funktion für den Zusammenprall der Parameter "details" verwendet werden. Dieser Parameter enthält alle Informationen, die über den Zusammenprall verfügbar sind, beispielsweise die Namen der beteiligten Starrkörper, den Punkt des Zusammenpralls und die relative Richtung des Zusammenpralls. Die letzte Information, die an die Callback-Funktion übermittelt wird, ist ein Wert, der die Stärke des Zusammenpralls angibt. Im oben stehenden Beispiel wird dieser Wert verwendet, um dem abzuspielenden Ton einen Lautstärkepegel zuzuordnen. Das bedeutet, dass ein härterer Aufprall zu einem lauteren Ton führt.
