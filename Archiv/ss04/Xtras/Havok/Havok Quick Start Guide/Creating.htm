Eine Havok-Simulation erstellen

Es gibt drei Möglichkeiten, eine physikalische Simulation mit Hilfe des Havok Xtra zu erstellen.

  1. Auf der Grundlage von W3D- und HKE-Dateien: Exportieren Sie aus einem externen Programm zum Erstellen von 3D-Modellen sowohl die 3D-Szene als W3D-Datei als auch die physikalischen Daten dieser Szene als Havok HKE-Datei. Auf diese Weise sind keine zusätzlichen Lingo-Skripts erforderlich.
  2. Auf der Grundlage einer W3D-Datei: Verwenden Sie einen Shockwave 3D-Darsteller, den Sie in einem externen Programm zum Erstellen von 3D-Modellen erstellt und als W3D-Datei gespeichert haben, und ordnen Sie den 3D-Modellen in der Shockwave 3D-Szene Starrkörper von Havok zu.
  3. Auf der Grundlage von Lingo: Alle Informationen für die dynamische 3D-Welt werden mit Hilfe von Lingo erstellt.
