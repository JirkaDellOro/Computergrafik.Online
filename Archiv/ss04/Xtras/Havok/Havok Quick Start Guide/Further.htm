Weitere Details

Diese Beispielszene steht nur für einen winzigen Teil der Leistungsfähigkeit des Havok Xtra. Während bisher alles auf der Grundlage von Lingo-Skripts erstellt wurde, können HKE-Dateien auch aus einem Programm zum Erstellen von 3D-Modellen exportiert und als Darsteller importiert werden. Dies geschieht in ähnlicher Weise bei Shockwave 3D-Dateien (*.W3D), die aus vielen Programmen zum Erstellen von 3D-Modellen exportiert werden können.

So können Sie im Handumdrehen Inhalte für physikalisch korrekte 3D-Szenen erstellen. Details zu Plug-Ins für Programme zum Erstellen von 3D-Modellen finden Sie auf der Website von Havok.

Doch werden nicht nur exportierte Dateien unterstützt, es werden auch zahlreiche Verhalten mitgeliefert. Auf diese kann in Macromedia Director auf dieselbe Weise zugegriffen werden und sie können auf dieselbe Weise verwendet werden wie alle anderen verfügbaren Verhalten. Zwei Gruppen sind dabei zu unterscheiden: "Setup" und "Control".

Die Verhalten vom Typ "Setup" umfassen die grundlegenden Funktionen, die in dieser Kurzanleitung beschrieben wurden. Initialisierung, Stepping und Beendigung werden von den Verhalten Physics (HKE) und Physics (No HKE) ausgeführt, während andere Verhalten für das Verwenden von Starrkörpern oder das Erstellen von Federn und Stoßdämpfern verantwortlich sind.

Die Verhalten vom Typ "Control" stellen Beispiele für komplexere Wechselbeziehungen in einer nach den Gesetzen der Physik simulierten Szene bereit. Das Verhalten Move Model ermöglicht es beispielsweise, Starrkörper mit Hilfe der Maus zu beeinflussen, während mit Drive Model jeder Starrkörper wie ein Auto gesteuert werden kann.

All diese Verhalten können Sie nach Belieben verwenden, untersuchen und verändern.
