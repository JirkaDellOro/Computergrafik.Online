W3D- und HKE-Dateien verwenden

Angenommen, eine Szene wurde bereits in einem externen Programm zum Erstellen von 3D-Modellen erstellt und sowohl eine W3D-Datei mit den 3D-Daten als auch eine HKE-Datei mit den physikalischen Daten exportiert. (Weitere Informationen zur Vorgehensweise beim Exportieren finden Sie in der Dokumentation zu Shockwave3D Exporter und Havok Exporter.)

Zunächst ist es erforderlich, die 3D-Welt zu importieren:

  1. Klicken Sie auf das Menü Datei.
  2. Wählen Sie Importieren im Menü Datei.
  3. Wählen Sie im Dialogfeld die gewünschte Shockwave 3D-Datei (*.W3D) aus.

Dadurch wird der 3D-Darsteller zur Besetzung hinzugefügt. Diese kann nun auf die Bühne gezogen werden. Importieren Sie nun den gewünschten Havok-Darsteller:

  1. Klicken Sie auf das Menü Datei.
  2. Wählen Sie Importieren im Menü Datei.
  3. Wählen Sie im Dialogfeld die gewünschte Havok-Exportdatei (*.HKE) aus.

Dadurch wird der Havok-Darsteller zur Besetzung hinzugefügt. Wenn Sie den Film in diesem Stadium abspielen, wird eine statische Welt angezeigt. Damit daraus eine dynamische Welt wird, muss die Simulation von Havok initialisiert und mit der 3D-Welt verbunden werden.

Um dies zu erreichen, ziehen Sie das Verhalten Physics (HKE) (das Sie in der Verhaltensbibliothek unter Havok > Setup finden) auf das Sprite, das eine Ansicht der 3D-Welt darstellt. Stellen Sie sicher, dass als Parameter Which Havok Cast Member der Name des Darstellers der Havok Physics-Szene gesetzt ist. Durch dieses Verhalten wird die Havok-Simulation initialisiert (3D-Objekte aus der W3D-Datei werden mit Starrkörpern in der HKE-Datei verbunden). Außerdem wird dafür gesorgt, dass die Zeit in der Physiksimulation nach jedem Schritt des Films fortschreitet und die Positionen und Ausrichtungen der aktuell gesteuerten 3D-Objekte aktualisiert werden. Sie können auch andere Parameter angeben, wenn Sie das Verhalten auf die Bühne ziehen - die standardmäßigen Einstellungen sind jedoch in den meisten Fällen sinnvoll.

Nun können weitere Verhalten auf die Bühne gezogen werden, um zusätzliche physikalische Effekte zu erzielen. Sie können beispielsweise das Verhalten Move Model (das Sie in der Verhaltensbibliothek unter Havok > Control finden) auf die Bühne ziehen und für den Parameter Which Model die Option Any Model auswählen. Ziehen Sie daraufhin einen Auslöser vom Typ Mouse Left auf die Bühne (aus der Verhaltensbibliothek 3D> Triggers) und ordnen Sie den Auslöser dem Verhalten Move Model zu (auswählbar im Parameter Select a Group and its Action).

Nun sollte es möglich sein, die 3D-Modelle in der Szene hochzuheben und mit Hilfe des Mauszeigers und der linken Maustaste zu verschieben.
