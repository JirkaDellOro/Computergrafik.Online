W3D-Datei verwenden

In diesem Beispiel ist keine Havok-Exportdatei (*.HKE) vorhanden, die importiert werden könnte. Also müssen die physikalischen Informationen mit Hilfe von Lingo erstellt oder unmittelbar aus der 3D-Welt gezogen werden.

Zunächst ist es erforderlich, die 3D-Welt zu importieren:

  1. Klicken Sie auf das Menü Datei.
  2. Wählen Sie Importieren im Menü Datei.
  3. Wählen Sie im Dialogfeld die gewünschte Shockwave 3D-Datei (*.W3D) aus.

Dadurch wird der 3D-Darsteller zur Besetzung hinzugefügt. Diese kann nun auf die Bühne gezogen werden. So erstellen Sie eine leere Havok-Simulation:

  1. Klicken Sie auf das Menü Einfügen.
  2. Wählen Sie Mediaelement im Menü Einfügen.
  3. Wählen Sie Havok Physics Scene im Untermenü Mediaelement.

Dadurch wird eine leerer Havok-Darsteller zur Besetzung hinzugefügt. Fügen Sie nun ein Physics (No HKE)-Verhalten zur Besetzung hinzu (aus der Verhaltensbibliothek Havok > Setup). Wenn Sie den Film in diesem Stadium abspielen, wird eine statische Welt angezeigt, das heißt, dass bisher den 3D-Modellen in der W3D-Datei noch keine Starrkörper zugeordnet sind.

Nun ist es erforderlich, dass Sie die Havok-Simulationsinformationen erstellen, indem Sie das Verhalten Physics (No HKE) bearbeiten. Die folgende Lingo-Funktion erstellt zwei Starrkörper, die zur Havok-Simulation hinzugefügt und den zwei 3D-Modellen "Box01" und "Text01" aus der W3D-Datei zugeordnet werden. Das erste davon ist ein fester konvexer Starrkörper, es darf sich also während der Simulation nie bewegen. Das zweite ist ein bewegliches konkaves Objekt. Eine detaillierte Beschreibung dieser Funktionen finden Sie im "Havok Xtra-Lingo-Referenzhandbuch".

on createScene

s = member( "HelloWorld" )
hk = member( "BlankHavok" )

-- create ground
m = s.model("Box01")
m.addModifier(#meshdeform)
hk.makeFixedRigidBody(m.name)

-- create text
m = s.model("Text01")
m.addModifier(#meshdeform)
hk.makeMovableRigidBody(m.name, 100.0, false)

end

Zu diesem Zeitpunkt können Sie nicht nur die Starrkörper erstellen, sondern Sie können auch deren standardmäßige physikalische Parameter ändern. Wenn Sie den Film nun abspielen, sollte eine physikalische Animation erfolgen (d. h., das Objekt "Text01" sollte unter Einwirkung der Schwerkraft herunterfallen).
