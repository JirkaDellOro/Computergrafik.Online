Lingo verwenden

In diesem Beispiel sind keine Havok-Exportdatei (*.HKE) und keine Shockwave 3D-Datei (*.W3D) vorhanden, die importiert werden könnten. Also müssen die Informationen zur Darstellung und zu den physikalischen Daten unmittelbar in Lingo erstellt werden.

Zunächst ist es erforderlich, die leere 3D-Welt zu erstellen:

  1. Klicken Sie auf das Menü Einfügen.
  2. Wählen Sie Mediaelement im Menü Einfügen.
  3. Wählen Sie Shockwave 3D im Untermenü Mediaelement.

Dadurch wird der 3D-Darsteller zur Besetzung hinzugefügt. Diese kann nun auf die Bühne gezogen werden. So erstellen Sie eine leere Havok-Simulation:

      1. Klicken Sie auf das Menü Einfügen.
      2. Wählen Sie Mediaelement im Menü Einfügen.
      3. Wählen Sie Havok Physics Scene im Untermenü Mediaelement.

Dadurch wird ein leerer Havok-Darsteller zur Besetzung hinzugefügt. Fügen Sie nun ein Physics (No HKE)-Verhalten (aus der Verhaltensbibliothek Havok > Setup) zur Besetzung hinzu, indem Sie es auf die Bühne ziehen. Stellen Sie sicher, dass als Parameter Which Havok Cast Member der Name des Darstellers der Havok Physics-Szene gesetzt ist.

Nun ist es erforderlich, die Physikszene und die 3D-Szene vollständig neu zu erstellen.

Die folgende Lingo-Funktion erstellt die 3D-Modelle für die Szene und die Starrkörper. In diesem Fall werden das 3D-Objekt, das den Boden darstellt, und die Textstarrkörper eigentlich als axial ausgerichtete Begrenzungsboxen konstruiert.

on createScene

s = member( "BlankScene" )
hk = member( "BlankHavok" )
txt = member( "Text" )

s.resetWorld()

-- create ground
mr = s.newmodelresource("GroundPlaneRes", #box)
mr.width = 150
mr.length = 50
mr.height = 100
m = s.newmodel("GroundPlane", mr)
m.transform.rotation = vector(-90,0,0)
m.transform.position = vector(0,250,-200)

m.addModifier(#meshdeform)
hk.makeFixedRigidBody(m.name, true, #box)

-- point camera
c = s.camera[1]
c.transform.position = vector(50,-50,-100)
c.pointat(m.transform.position + vector(0,0,50), vector(0,0,1))
c.hither = 1
c.yon = 1000

-- create text
txt.text = "Hello World"
mr = txt.extrude3D( s )
mr.bevelType = 1
mr.bevelDepth = 0.1
mr.smoothness = 6
m = s.newModel("Text", mr)
m.transform.position = vector(-75,250,0)
m.transform.rotation = vector(90,0,0)
m.addModifier(#meshdeform)
hk.makeMovableRigidBody(m.name, 1000.0, true, #box)

end
